# Jal-Sanchay Tracker 💧

Jal-Sanchay Tracker is an Android application designed to promote sustainable water habits. It tracks real-time water conservation data and helps users visualize their usage patterns.

## 🌟 Key Features
* Real-Time Tracking: Monitor your water conservation efforts as they happen.
* Usage Visualization: Understand your water usage patterns through clear visual data.
* Goal Setting: Set personalized water-saving goals to stay motivated.
* Water Wealth Conversion: Input your roof area and local rainfall to calculate and convert rainwater harvesting into measurable "Water Wealth."  Logic: Formula: Area x Rainfall x 0.0929 x Runoff Coefficient.

## 🛠️ Tech Stack
* Platform: Android
* Language: Kotlin / Java
* UI/Architecture: Jetpack
